
# 贪吃蛇

为fx-991cnx(**ver C**)设计的贪吃蛇程序

## 已知问题

这是为fx-991cnx设计的第一个ROP程序，因此有较多问题
- 由于选择了较为简单的绘图方式以及设置了不合理的数据储存地址，蛇的最大长度受到很大限制，超出长度限制后会导致白屏
- 随机数生成方法较为简单，有概率将点位生成在屏幕外
- 绘图函数的优化问题导致蛇较长时运行缓慢
- 所选择的按键检测函数有概率因计时器溢出而卡死。按AC键可解除
- 未使用DI指令屏蔽中断，因此程序有概率卡死

## 效果

- 蛇由一系列连续的3x3方块构成，可在屏幕上运动。方向键控制蛇的运动方向
- 规则与常规贪吃蛇游戏一致
- 无计分系统

## 注入

1. 使用内存写入程序（或其他方法）将`snake1.5.bin`内容写入`0xE9E0`处
2. 将`data_1.5.bin`内容写入`0xD9F0`处
3. 在`0xEEA0`处写入如下数据:

```
30 80 AE EE A4 EE 03 00 06 00 B6 EE BC EE 00 00
09 09 09 0A 09 0B 0C 09 0C 0A 0C 0B 0C 09 0C 0A
0C 0B C6 D9
```

4. 切换为小字体，输入`launcher.bin`内容，按等号执行程序
